集群软件

目前我们的集群安装了如下的软件，用户需根据自己的需求进行载入。绝大多数软件都可以使用

module add <module name>

进行载入。使用

module help <module name>

可以查看对应软件模块的详细信息，包括内嵌软件版本以及可能的冲突等。加粗的版本 是使用 module add 命令不指定版本号时载入的默认版本；用‘‘*’’标出的是登录系统 后自动载入的版本。
